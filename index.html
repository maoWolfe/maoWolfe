HTML Cheatsheets

Table Tags
Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tfoot> Table Foot The set of rows defining the footer in a table.
<th> Table Heading Add titles to rows and columns of a table and must be enclosed in a table row element, <tr>.
<tr> Table Row Add rows to a table before adding table data and table headings.
<td> Table Data Add a cell of data to a table.
Table Attributes
Attribute Name Description
<scope> Scope Specifies whether a header cell is a header for a column, row, or group of columns or rows.
<colspan> Column Span Defines how many columns a td element should span.
<rowspan> Row Span Defines how many rows a td element should span.
